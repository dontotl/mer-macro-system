# 영업 업무자동화 계획

작성일: 2026-04-01

## 목표
다양한 세일즈 채널에서 재사용 가능한 기술/사례/교육 콘텐츠를 빠르게 찾고, 최신 정보로 클라이언트 맞춤 자료를 만드는 운영체계를 구축한다.

## 핵심 자동화 영역

### 1. 경쟁사/시장 리서치 자동화
- 웹검색을 통한 경쟁사 뉴스/제품/가격/파트너십 모니터링
- 변경점만 요약하는 데일리/위클리 브리프 생성
- 키워드 세트: 회사명, 제품명, 기술명, 규제/파트너십/출시 관련 표현
- 결과 저장: 날짜별 md 브리프 + 주제별 요약 문서

### 2. 내부 자료 문맥검색(RAG)
- 외장하드 내 PPT/Word/XLSX/PDF 문서 색인
- 텍스트 추출 → 청크 분할 → 임베딩 → 문맥검색
- 메타데이터: 고객사, 산업군, 제품군, 작성일, 원본 경로
- 검색 출력: 답변 + 출처 문단 + 원본 파일 경로

### 3. 클라이언트 맞춤 발표 자동화
- 공통 모듈(시장/경쟁/기술/ROI/도입 로드맵) 템플릿화
- 고객 산업/페르소나(임원/실무/기술팀) 별 버전 생성
- 최신 경쟁사/시장 동향을 자동 삽입하는 발표 초안 생성
- 발표 외 파생물: 1페이지 요약, 이메일 초안, 미팅 후속 메모

### 4. SharePoint / Seismic 운영
- SharePoint: 내부 위키, 원본/초안/리뷰 협업 공간
- Seismic: 고객 전달용 승인 콘텐츠 저장소
- 콘텐츠 축:
  - 기술(Technical)
  - 사례(Use Case)
  - 교육(Enablement)
- 메타데이터 필수 항목: Owner, Last Updated, 승인 상태, 대상 산업, 대상 채널

## 운영 루틴
- Daily: 경쟁사 변화 5줄 브리프
- Weekly: 시장/기술 업데이트 1페이지 요약
- Per Client: 맞춤 발표 초안 생성 및 최신성 점검
- Monthly: 오래된 자료 폐기/갱신 점검

## 우선 구현 순서
1. 투자/리서치 md 파일 운영 안정화
2. 문서 색인 파이프라인(PDF/PPTX/DOCX/XLSX)
3. 검색 UI 또는 질의 템플릿
4. PPT/제안서 생성 템플릿
5. SharePoint / Seismic 업로드 운영안 정교화
