# PRD — KRX RS 추세추종 전략 백테스트/실행 규칙

버전: v1.0
작성일: 2026-04-01
상태: 초안

## 1. 목적
코스피 대비 상대강도(RS)가 높은 대형주(시총 1조+) 중 추세가 강한 종목을 선별해,
단순 유닛 매수/불타기/손절 규칙으로 일관된 매매를 수행한다.

## 2. 유니버스 & 데이터
### 2.1 유니버스
- 거래소: KOSPI + KOSDAQ
- 시가총액: 1조 원 이상
- 데이터 소스: pykrx (일봉 OHLCV, 시총, 지수)

### 2.2 계산 지표
- 이동평균선: MA5, MA10, MA20, MA60
- 3개월 신고가: 최근 60거래일 기준
- RS(상대강도): 종목 60거래일 수익률 vs 코스피 60거래일 수익률

## 3. 진입 조건
아래 조건을 모두 만족해야 한다.
1. 시가총액 1조 이상
2. RS 상위군 (기본값: 상위 20%)
3. 3개월 신고가
4. 정배열: MA5 > MA10 > MA20 > MA60

## 4. 포지션 규칙
- 계좌 5분할
- 1유닛 = 총자산의 20%
- 종목당 최대 3유닛

### 최초 진입
- 조건 충족 시 1유닛 매수

### 불타기
- 기준가: 최초 진입가
- 최초 진입가 대비 +8% 도달 시 1유닛 추가
- 최초 진입가 대비 +16% 도달 시 1유닛 추가
- 같은 단계는 중복 실행 금지

## 5. 청산 규칙
### 5.1 추세 이탈
- 종가가 MA20 아래로 내려가면 전량 매도

### 5.2 -8% 손절
- 1유닛 보유: 1유닛 진입가 대비 -8% 시 전량 매도
- 2유닛 보유: 2유닛 진입가 대비 -8% 시 전량 매도
- 3유닛 보유: 3유닛 진입가 대비 -8% 시 전량 매도

즉, 손절 기준은 항상 마지막으로 추가된 유닛의 진입가다.

## 6. 체결 규칙
- 신호 판단: 당일 종가 기준
- 체결: 다음 거래일 시가 기준
- 거래비용: 수수료/슬리피지 반영
