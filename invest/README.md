# 투자 노트 인덱스

이 폴더는 투자 관련 정보를 날짜/주제 단위로 누적 정리하는 공간입니다.

## 운영 원칙
- 사실/출처/의견을 분리해서 기록
- 확정 정보와 루머를 구분
- 시장 일정은 한국시간(KST) 기준으로 우선 표기
- 투자 판단은 최종적으로 사용자 본인 책임

## 파일 구조
- `daily-brief-YYYY-MM-DD.md`: 일일 지표/시황 요약
- `macro-calendar-us-kr.md`: 미국/한국 주요 매크로 일정 캘린더
- `aacr-2026-watchlist.md`: AACR 2026 관련 관심 종목/체크포인트
