# 미국·한국 주요 매크로 일정 (관리용)

- 기준: KST(Asia/Seoul)
- 목적: 주간/월간 변동성 이벤트 사전 체크
- 업데이트 규칙: 일정 확정 시 수시 갱신

## 미국 (US)
- 고용: 비농업고용(NFP), 실업률, 임금상승률
- 물가: CPI, PPI, PCE
- 경기: ISM 제조업/서비스업 PMI, 소매판매, 산업생산
- 통화정책: FOMC 금리결정, 점도표, 파월 기자회견
- 기타: 국채입찰(2Y/10Y/30Y), 재무부 분기 차입계획

## 한국 (KR)
- 물가: 소비자물가(CPI)
- 경기: 수출입(월간), 산업활동동향, 고용동향
- 통화정책: 한국은행 금통위 기준금리 결정
- 심리지표: 소비자심리지수, 기업경기실사지수(BSI)
- 기타: 기재부/한은 주요 브리핑 일정

## 활용 방법
1. 이벤트 24~48시간 전: 컨센서스/직전치 확인
2. 발표 직후: 헤드라인 수치 vs 컨센서스 괴리 점검
3. 시장 반응: 금리(미국채/국고채) → 환율 → 주가지수 순으로 확인
4. 사후 정리: 결과가 기존 내러티브(침체/연착륙/리플레이션)와 맞는지 기록
