# Mer Macro Outputs Reference

## Daily
- markdown: `invest/notes/daily-macro/YYYY-MM-DD.md`
- telegram text: `invest/notes/daily-macro/YYYY-MM-DD.telegram.txt`
- chart: `invest/notes/daily-macro/charts/YYYY-MM-DD-liquidity.png`

## Weekly
- markdown: `invest/notes/daily-macro/weekly/YYYY-Www.md`
- telegram text: `invest/notes/daily-macro/weekly/YYYY-Www.telegram.txt`

## Included summary blocks
- Liquidity
- Stress
- Inflation
- Money Scale
- Repo Summary (SOFR/EFFR/spread)
- Policy Summary (Fed target band + WALCL)
