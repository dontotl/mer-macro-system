# Evaluation Rubric

Use this rubric to avoid turning weak source material into overconfident notes.

## 1. Thesis
Ask:
- What is the single-sentence investment thesis?
- Is this about earnings, valuation, policy, cycle, supply chain, or sentiment?
- Is the time horizon short, medium, or long?

## 2. Evidence
Separate evidence into:
- reported data
- company disclosures
- channel checks / anecdotal points
- modeled projections
- macro assumptions

Weight reported data above modeled projections.

## 3. Speculation markers
Treat the following carefully:
- TAM-heavy arguments without near-term revenue bridge
- policy benefits not yet implemented
- capacity expansion without utilization proof
- target price claims without robust operating bridge
- “platform” or “ecosystem” language without unit economics

## 4. Note-writing rules
- Do not promote the source; interpret it
- Preserve key numbers exactly when possible
- Mark uncertainty plainly
- If one business segment is clearly strongest, say so
- If one segment is mostly optionality, say so

## 5. Practical close
End with:
- what to monitor next
- what would confirm the thesis
- what would break the thesis
