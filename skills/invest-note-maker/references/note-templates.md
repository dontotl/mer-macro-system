# Note Templates

## 1. Detailed company note template

```md
# <회사명/주제>

- 작성일: <YYYY-MM-DD>
- 출처: <source>
- 성격: 종목분석 / 산업리포트 요약노트
- 종목: <회사명/티커>
- 태그: #태그1 #태그2 #태그3

## 한줄 요약
<한 문장으로 투자 아이디어>

## 투자포인트 요약
1. <핵심 포인트>
2. <핵심 포인트>
3. <핵심 포인트>

## 산업 리포트형 정리
### 1) <산업 축>
- <요약>

### 2) <회사 축>
- <요약>

### 3) <옵션/신사업 축>
- <요약>

## 기업 분석 핵심
- 사업 구조:
- 주요 이익원:
- 밸류체인 포지션:
- 경쟁력:

## 숫자로 보는 보고서 핵심
- 투자의견:
- 목표주가:
- 현재주가:
- 상승여력:
- 핵심 실적 추정:

## 해석: 이 자료를 어떻게 읽을까
### 좋은 점
- <bullet>

### 조심할 점
- <bullet>

### 실전적으로 보면
- <bullet>

## 체크리스트
- [ ] <확인 항목>
- [ ] <확인 항목>

## 검색용 키워드
- <keyword>
- <keyword>

## 원문 성격 메모
<sell-side / buy-side / 학생리포트 / 기사 / 브로커콜 요약 등>
```

## 2. Detailed industry note template

```md
# <산업/테마명>

- 작성일: <YYYY-MM-DD>
- 출처: <source>
- 성격: 산업리포트 요약노트
- 산업/테마: <sector/theme>
- 태그: #태그1 #태그2 #태그3

## 한줄 요약

## 산업 핵심 포인트
1. <수요>
2. <공급>
3. <정책/가격/사이클>

## 밸류체인 정리
- Upstream:
- Midstream:
- Downstream:

## 수혜/피해 포지션
- 수혜 가능:
- 피해 가능:

## 숫자로 보는 핵심
- <숫자>
- <숫자>

## 해석
### 구조적 포인트
- <bullet>

### 단기 변수
- <bullet>

### 추적할 지표
- <bullet>

## 체크리스트
- [ ] <지표>
- [ ] <정책>
- [ ] <가격>
```
