# mer-macro-system

메르 매크로 노트 자동 생성(일간/주간), 텔레그램 포맷 텍스트, 유동성 차트까지 한 번에 만드는 OpenClaw skill입니다.

## 핵심 기능
- Daily macro note (`.md`)
- Daily telegram digest (`.telegram.txt`)
- Repo summary (SOFR/EFFR/spread)
- Policy summary (Fed target band + WALCL)
- Weekly summary (`.md` + `.telegram.txt`)

자세한 설치/활용은 `docs/` 참고:
- `docs/INSTALL.md`
- `docs/USAGE.md`
